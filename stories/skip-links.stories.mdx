# Skip Links

<Subtitle>
	Provide ways for users to skip to important sections of our websites. It helps users using screen-readers to navigate
	easier and more efficiently.
</Subtitle>

## Usage

Press the <kbd>`tab`</kbd> key and you'll see that there is only one skip link being shown.
Press <kbd>`enter`</kbd> when it is visible and you'll set focus on the area of that `id`.
